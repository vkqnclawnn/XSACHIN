🎯 서비스 소개

X사친 테스트는 연인들의 질투심 수준을 재미있게 측정하고 서로 비교할 수 있는 온라인 심리 테스트입니다.

✨ 주요 기능

  • 10가지 질문으로 나의 질투심 레벨 측정
  • 연인과의 결과 비교 기능
  • 각 질문별 답변 상세 비교
  • 사귄 날짜 입력으로 더 재미있는 분석
  • 소셜미디어 공유 최적화

💝 왜 만들었나요?

연인 사이에서 "어디까지가 괜찮고, 어디서부터 질투를 느끼는지"는 항상 궁금한 주제입니다. 서로의 마음을 더 잘 이해하고, 재미있게 소통할 수 있는 도구를 만들고 싶었습니다.

"우리 둘 중 누가 더 질투쟁이일까? 궁금하지 않나요? 😉"

🛡️ 개인정보 보호

사용자의 개인정보 보호를 최우선으로 생각합니다:

  • 별도의 회원가입이나 개인정보 입력 불필요
  • 테스트 결과는 30일 후 자동 삭제
  • 제3자와 정보 공유 금지
  • 안전한 HTTPS 연결로 데이터 보호
